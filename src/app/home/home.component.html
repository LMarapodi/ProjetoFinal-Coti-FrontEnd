Pagina Incial

Proposta :

- Criar um Projeto Api com rotinas de Cadastro,Consultas,Atualizações e Exclusão de Atendimentos que poderia ser utilizado em um hospital.


Regras de Negocios :

- Pacientes novos não podem possuir Cpf ou Email já contidos no Banco de Dados.

- Paciente ou Medico só podem ser excluidos caso não existam atendimentos marcados.


Tecnologias Utilizadas :
Back-End

- Net Core Api.

- Swagger.

- C#.

- DDD (Domain Drive Design).

- Entity FrameWork.

Front-End

- Angular.

- Html, Css e JavaScript.

- Bootstrap.
